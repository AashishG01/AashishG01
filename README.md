### 🚀 Featured Project: Video Brain (VRAG)

**[Video Brain (VRAG)](https://github.com/AashishG01/VRAG)**
A complete Video-based Retrieval-Augmented Generation (VRAG) pipeline built entirely from scratch. 
- **Tech:** Python, FastAPI, React, Qdrant, FFmpeg, Whisper, CLIP, BGE, Groq (LLaMA 3 70B & LLaMA 4 17B)
- **Features:** Smart frame extraction via scene detection, word-level audio transcription, hybrid cross-modal search (visual + text), and multi-modal VLM reasoning to chat with video content natively.
